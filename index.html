VectorShop: Semantic Product Search System

A production-ready semantic search engine for e-commerce that understands natural language queries

VectorShop is a sophisticated semantic search system for small-to-medium sized online stores, allowing customers to find products using natural language queries instead of just keywords. The system combines keyword search, vector similarity, and AI reasoning to deliver highly relevant search results even for complex queries.

The system uses a hybrid architecture that combines sparse retrieval (BM25) with dense vector embeddings (FAISS) and a reranking layer using transformer models. This approach delivers sub-100ms query times even with millions of products, making it suitable for production environments.

Key Features

  • Natural language understanding for complex product queries
  • Hybrid search technology combining keywords, vectors, and AI reasoning
  • Cost-effective implementation using optimized open-source models
  • Multilingual support for both English and regional languages
  • Easy integration with popular e-commerce platforms like Shopify

Technologies Used

Python 3.9+ PyTorch 1.9+ FAISS Transformers DeepSeek-R1-Distill BM25 Natural Language Processing Vector Search

SpaceX Launch Success Prediction

Machine learning project predicting Falcon 9 first stage landing success

This project uses machine learning to predict SpaceX Falcon 9 first stage landing success, which helps estimate launch costs since SpaceX rocket launches are significantly cheaper when the first stage can be reused. The analysis combines data from multiple sources, including the SpaceX REST API and web scraping from Wikipedia.

The project achieved 83% prediction accuracy by analyzing launch records with details on boosters, payloads, and landing outcomes. It includes interactive visualizations and dashboards to explore the data and understand the factors that influence landing success.

Key Features

  • Comprehensive data collection from SpaceX API and web scraping
  • Interactive dashboards and geographic visualizations
  • Multiple machine learning models including Logistic Regression, SVM, Decision Trees, and KNN
  • 83% prediction accuracy on test data
  • Analysis of key factors influencing launch success

Technologies Used

Python 3.x Pandas NumPy Scikit-learn Matplotlib Seaborn Plotly Dash Folium API Integration Web Scraping

AI Blockchain Security Analysis System

Chainlink Hackathon 2024 project for analyzing blockchain security

The Generative AI Blockchain Security Analysis System leverages artificial intelligence to analyze blockchain transactions and smart contracts for security vulnerabilities and potential fraud. This project was developed for the Chainlink Hackathon 2024 and implements a chain-agnostic approach supporting Ethereum, Polygon, and Avalanche networks.

The system combines Chainlink Functions for secure on-chain data retrieval with OpenAI's LLM for security analysis, providing detailed reports on potential vulnerabilities and suspicious transactions. Analysis reports are stored on IPFS via QuickNode for decentralized, permanent storage.

Key Features

  • Smart contract vulnerability analysis with severity categorization
  • Blockchain transaction fraud detection across multiple chains
  • Chain-agnostic implementation supporting Ethereum, Polygon, and Avalanche
  • IPFS integration for decentralized storage of analysis reports
  • AI-powered fraud detection with confidence ratings

Technologies Used

Node.js & Express TypeScript Angular Chainlink Functions OpenAI API IPFS via QuickNode BitQuery Ethereum Polygon Avalanche
